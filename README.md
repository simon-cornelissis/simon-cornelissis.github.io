# Eindhoven International Project Presentation

This repository contains the Reveal.js-based presentation for the international project in Eindhoven.

## How to Publish to GitHub Pages

1. Go to the **Settings** tab of this repository on GitHub.
2. Select **Pages** from the left sidebar.
3. Under **Build and deployment**, ensure the source is set to **Deploy from a branch**.
4. Select the **main** (or your primary) branch and the **/(root)** folder.
5. Click **Save**.
6. After a few minutes, your presentation will be live at `https://<your-username>.github.io/<repository-name>/`.

## Features

- Built with [Reveal.js](https://revealjs.com/)
- Responsive design
- Custom theme matching project colors
- Focus on AI in Transport & Delivery
